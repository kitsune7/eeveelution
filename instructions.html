Create a Boot Disk to Reset Your Windows Password

Hopefully you've never had to experience the trauma of forgetting a password to a computer. Obviously, it's not the end of the world when it happens, and there are ways of getting around a login password without any harm to you or your computer. In these instructions, we'll show you how to create a boot disk on a flash drive that will let you reset any account's password on your Windows computer.

Tools

You'll need:

  • USB drive
  • Windows computer or laptop
  • Administrator account access

Steps

  1. Download the Boot Disk Creator

    2. Click here to download Password_Reset_Creator.zip.

    Next, open up your downloads folder, right click 'Password_Reset_Creator.zip', and select "Extract All". Click the extract button when a popup comes up to confirm the location to extract it to.

  2. Run 'createdrive.bat' as administrator

    3. In the newly opened folder, double click the 'Password_Reset_Creator' folder, right click 'createdrive', and select "Run as Administrator".

    Select "Yes" at the prompt.

  3. Confirm the flash drive to use

    4. If you have more than one USB drive inserted, you will be prompted to choose which one to erase. The script will then ask you to confirm your choice.

    Caution! All data will be erased from the flash drive! Please backup any data on the flash drive beforehand, and make sure you are using the correct drive.

    Once you type 'Y' for yes, press the enter key and watch the script run.

  4. Exit the program and restart Windows

    5. The script is finished when it says "Press any key to exit..."

    Exit the window and shut down the computer. You've now built the boot disk and put it on a flash drive. The next steps will show you how to use it.

  5. Enter boot menu

    6. Once you see the screen go black, repeatedly press the Escape button on your keyboard. Some computers may require a different hotkey. Regardless of which key is required, there should be a message like "Press ESC for Boot Menu" that tells you which key to press.

    If it boots into Windows, reboot and try again. If it works, you'll see list of choices to boot from.

  6. Boot from USB drive

    7. Highlight the USB drive and press Enter. The USB drive may also be called a removable device.

    If all goes well, you'll see the loading screen.

    The menu will come up next. Press Enter to edit user data and passwords.

  7. Select an RID

    8. On this screen, you'll see a list of usernames and RIDs. Type in the RID to the left of the username you want to reset the password for. In this case, we want to reset the password for alyssa's account, so we type in "03eb".

  8. Clear the password

    9. Next, enter 1 to clear the user's password.

  9. Quit and write hive files

    10. Now that the user's password is cleared, we can exit and reboot the computer. Press 'q' in the User Edit Menu and Main Menu to quit the program.

    When it asks to "Write hive files?", make sure to enter 'y'! Otherwise, your changes will not be saved.

  10. Reboot Windows and login

    11. Finally, reboot Windows. Now you can login without a password to the account you edited! Congratulations!

Conclusion

If you were able to successfully login to the previously password protected user, then congratulations! You did it! If you have a random PC with a locked account, you will be able to use this tool to gain access to login privileges. This could be extremely useful when a you or a friend forgets their password, when purchasing used computers, or in any other Windows login situations. If you weren't before, you should now be more familiar with the PC bootup process.
