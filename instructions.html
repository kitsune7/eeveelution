Create a Boot Disk to Reset Your Windows Password

Hopefully you've never had to experience the trauma of forgetting a password to a computer. Obviously, it's not the end of the world when it happens, and there are ways of getting around a login password without any harm to you or your computer. Today we are going to bestow this sacred power into your mortal hands.

You'll need:

  • USB drive
  • Windows computer or laptop
  • Administrator account access

  1. Download Password_Reset_Creator.zip

    2. Open up a web browser and navigate to http://tinyurl.com/ENGL316-pw. This will download Password_Reset_Creator.zip.

  2. Extract the contents of the zip file

    3. Open up your downloads folder, right click 'Password_Reset_Creator.zip', and select "Extract All". Click the extract button when a popup comes up to confirm the location to extract it to.

  3. Run 'createdrive.bat' as administrator

    4. In the newly opened folder, right click 'createdrive.bat' and select "Run as Administrator".

    Select "Yes" at the prompt.

  4. Confirm the flash drive to use

    5. If you have more than one USB drive inserted, you will be prompted to choose which one to erase. The script will then ask you to confirm your choice.

    Caution! All data will be erased from the flash drive! Please backup any data on the flash drive beforehand, and make sure you are using the correct drive.

    Once you type 'Y' for yes, press the enter key and watch the script run.

  5. Exit the program and restart Windows

    6. The script is finished when it says "Press any key to exit..."

    Exit the window and shut down the computer. You've now built the boot disk and put it on a flash drive. The next steps will show you how to use it.

  6. Boot from the USB drive

    7. When you turn your computer back on, look at your screen for an option that says "Boot Options", "Boot Menu", or "Startup Device". The specific hotkey may differ depending on what computer you use, but the hotkey should be listed next to the options listed on screen. Press that button as the computer starts up.

    Next, choose to boot to the USB drive.

    If all goes well, you'll see this loading screen.

    This menu will come up next.

  7. Select an RID

    8. On this screen, you'll see a list of usernames and RIDs. Type in the RID to the left of the username you want to reset the password for. In this case, we want to reset the password for alyssa's account, so we type in "03eb".

  8. Clear the password

    9. Next, enter 1 to clear the user's password.

  9. Quit and write hive files

    10. Now that the user's password is cleared, we can exit and reboot the computer. Press 'q' in the User Edit Menu and Main Menu to quit the program.

    When it asks to "Write hive files?", make sure to enter 'y'! Otherwise, your changes will not be saved.

  10. Reboot Windows and login

    11. Finally, reboot Windows. Now you can login without a password to the account you edited! Congratulations!

And just like that, you're a personal computer system administrator. If you have a random PC that's account is locked, you can now get back in and use it like normal.
